# Flutter 3D Accessory

Ứng dụng Flutter thử nghiệm việc xem nhân vật 3D, gắn phụ kiện `.glb`, lưu cấu hình nhân vật và phát animation trong phòng 3D bằng WebView + Three.js.

## Chạy nhanh

```bash
flutter pub get
flutter run
```

## Tài liệu chi tiết

Xem thư mục [docs](docs/README.md) để đọc thêm về kiến trúc, luồng hoạt động, quy trình thêm asset, tối ưu hiệu năng và triển khai.
